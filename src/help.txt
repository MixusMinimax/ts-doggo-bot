`================[HELP]================`
> !help - this text
> !ping - measure ping
> !say <string> - deletes your message and says it itself.
> !info - info about DoggoBot!
> !time - displays the current time
> `deprecated` !links - links for the current channel
> `deprecated` !links \\n (<link>\\n)* - add new links to this channel
> `deprecated` !linkremove <int> - remove link at index i; remember: the headline of !links is index 0. So the first link is index 1
> !purge <int> - deletes the last n messages in this channel. WARNING: There's no warning.
